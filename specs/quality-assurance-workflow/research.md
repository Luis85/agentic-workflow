# Research — Quality assurance workflow baseline

## Sources

- ISO page for ISO 9001:2015 confirms the current published requirements edition and references ISO 9001:2015/Amd 1:2024.
- ISO page for ISO/FDIS 9001 confirms a future edition is under development in 2026.
- Existing repository tracks use command namespaces, reusable skills, templates, state files, sink documentation, generated command inventories, and traceable specs.

## Alternatives

- Add only a checklist template. Rejected because users need a guided workflow and corrective action loop.
- Add a full lifecycle stage. Rejected because QA reviews may apply to projects, portfolios, releases, suppliers, or active features, not only one lifecycle stage.
- Add an opt-in companion track. Chosen because it matches the existing track model and keeps lifecycle ownership intact.
