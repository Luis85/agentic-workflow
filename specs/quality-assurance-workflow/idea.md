# Idea — Quality assurance workflow

Users need a guided quality assurance workflow that checks whether project execution is healthy enough to deliver a quality product. The workflow should be ISO 9001-aligned, evidence-backed, and checklist-driven.

## Scope

- Add a Quality Assurance Track with slash commands.
- Add a reusable quality-assurance skill.
- Add templates for QA state, plan, checklist, review, and improvement plan.
- Update core docs and command inventories.

## Non-goals

- No claim of ISO certification.
- No reproduction of ISO standard text.
- No executable automation beyond workflow guidance and templates.
