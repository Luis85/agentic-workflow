# agentic-workflow

## Definition

`agentic-workflow` is the repository slug and GitHub Packages npm package identity for Specorator.

## Why it matters

The product is branded as Specorator, but users clone, fork, link, and install it through the `Luis85/agentic-workflow` repository and the `@luis85/agentic-workflow` package. ADR-0024 locks that distinction for v1.0 stability.

## Examples

```bash
git clone https://github.com/Luis85/agentic-workflow.git my-project
npm install --save-dev @luis85/agentic-workflow
```
