# Specorator

## Definition

Specorator is the product name for this spec-driven, agentic software-development workflow template.

## Why it matters

Specorator names the method, brand, product page, and product steering. The repository and package identity remain `agentic-workflow` under ADR-0024, so public surfaces use Specorator for the product and `agentic-workflow` for clone, install, package, and URL references.

## Examples

- *Specorator keeps humans in charge of what to build while agents handle how.*
- *Install the current template package as `@luis85/agentic-workflow`.*
